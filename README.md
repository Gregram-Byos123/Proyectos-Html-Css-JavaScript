## Proyectos de HTML, CSS y JS

El proyecto consiste en realizar un roadmap sobre el recorrido oficial vívido por mí en el desarrollo de estos lenguajes de programación en la web (Front-End).

## Descripción
Este proyecto permite conocer cómo funciona cada uno de estos lenguajes. Con ejemplos donde se presenta su código y resultado a los espectadores.  
Está pensado para programadores principiantes (como yo en algún momento) que deseen seguir una guía con fundamento.

## Tecnologías
- HTML
- CSS
- JavaScript

## Instalación
No requiere instalación.

1. Clona el repositorio.  
2. Abre el archivo `index.html` en el navegador.


## Uso
Explica en una o dos líneas cómo se usa el proyecto una vez ejecutado.

## Estructura
- `index.html` Es donde se guarda la estructura de las páginas.
- `style.css` Es las decoraciones hechas sobre la estructura (animaciones, interacciones, hovers, gradientes, colores, backgrounds...).
- `script.js` La parte funcional de la página/as dónde se ubican los algoritmos, funciones y parte totalmente funcional de las páginas webs.
